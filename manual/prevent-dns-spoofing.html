DNS over HTTPS

DOH（DNS over HTTPS）是一种将域名解析请求通过 HTTPS 协议加密传输的技术。通常情况下，DNS 查询是以纯文本形式传输的，容易被第三方截取和监控。DOH 则将 DNS 查询加密，混合在常规的 HTTPS 网络流量中，增加了用户隐私保护和安全性，减少了被篡改和拦截的风险。

DOH 的工作流程包括将用户的 DNS 查询请求包装在 HTTPS 请求中，发送到支持 DOH 的 DNS 服务器上，然后服务器解密并解析请求，返回加密的 IP 地址信息。这样，即使在公共网络中，也能有效保护用户的浏览隐私。但需要注意的是，DOH 对网络设备和服务器的资源要求较高，可能会影响性能。
