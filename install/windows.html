Windows安装v2rayA

发表于2021-08-31|更新于2024-12-24|安装教程Windows
|阅读量:

本文介绍如何在 Windows 上安装 v2rayA。需要注意的是，目前在 Windows 上仅支持一键配置系统代理而非透明代理。

1. 使用安装包

安装包将内置 v2ray-core，如需更换 Xray-core，可在安装完毕后于安装目录手动进行替换。

1.1. 方法一：通过 WinGet 自动安装

WinGet 是微软推出的软件包管理器，适用于 Windows 10 以及更新版本的操作系统。

1.2. 方法二：手动运行安装包

从 GitHub Releases 下载适用于 Windows 的安装包，例如 installer_windows_inno_x64_2.2.6.2.exe，按照指示安装完毕即可。如果github无法访问，或者速度很慢，可以通过ghproxy下载 installer_windows_inno_x64

2. v2rayA 使用方法

通过安装包安装 v2rayA 后，v2rayA 将以服务的形式运行，默认情况下将开机自启，你也可以在任务管理器中的“服务”选项卡管理 v2rayA 的启动与停止。你可以通过运行桌面快捷方式或直接访问 http://127.0.0.1:2017 打开管理页面。

v2rayA 具体使用方法可以访问 v2rayA 基本使用方法
