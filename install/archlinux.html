v2rayA Arch/Manjaro Linux 安装教程

发表于2020-11-16|更新于2024-11-14|安装教程Linux
|阅读量:

本文主要讲述如何在Arch/Manjaro Linux上安装v2rayA

1. 安装 V2Ray 内核

从官方源安装 v2ray。

2. 安装 v2rayA

从 AUR 安装 v2raya 或 v2raya-bin、v2raya-git 即可。

3. v2rayA 使用方法

安装并成功启动v2rayA服务后，可以使用浏览器直接访问 http://127.0.0.1:2017 v2rayA管理页面, 具体使用方法可以访问 v2rayA 基本使用方法
